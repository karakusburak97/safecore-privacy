## Veri Toplama
SafeCore uygulaması aşağıdaki verileri yalnızca cihazınızda saklar:
- Profil bilgileri (ad, soyad, yaş, boy, kilo, kan grubu)
- Sağlık bilgileri (hastalıklar, ilaçlar)
- Acil iletişim kişileri
- Kişisel notlar
